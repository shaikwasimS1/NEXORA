# 🚀 Creative Agency Website / Portfolio with three live projects

A modern, animated portfolio website built with Next.js and Tailwind CSS, featuring smooth UI interactions, 3D tilt cards, and a premium dark-themed design.

---

## 📌 Features

- ⚡ Modern responsive UI (mobile + desktop)
- 🎨 Premium dark aesthetic design
- 🧠 Interactive 3D tilt cards (Trending section)
- 🌈 Gradient text & glow effects
- 🖼️ Dynamic portfolio/trending image grid
- ✨ Smooth hover animations
- 🧩 Component-based architecture (Next.js)
- 📱 Fully responsive layout

---

## 🛠 Tech Stack

- **Next.js 13+ (App Router)**
- **React.js**
- **Javascript/TypeScript**
- **Tailwind CSS**
- **Framer Motion (animations)**
- **Lucide Icons**
- **Custom UI Components (Button, TiltCard, etc.)**
